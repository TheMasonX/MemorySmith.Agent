## Notes

String is obtained by killing spiders or cave spiders (0–2 string per kill),
or breaking cobwebs (1 string). It is also found in jungle temple and desert
temple loot. 4 string can be crafted into 1 wool.

**Gather strategy:**
- Hunt spiders at night or in caves
- Break cobwebs with a sword (fastest) or shears
- Fish for string as a junk item (rare)

**Crafting uses:**
- 4 string → 1 white_wool (alternative to shearing sheep)
- 3 string + 3 sticks → 1 fishing_rod
- 3 string + 2 sticks → 1 bow
- 2 string → 1 lead (with 1 slime_ball)
- Tripwire hooks and tripwire circuits

**Related items:** `white_wool`, `bow`, `fishing_rod`, `lead`
