About

MemorySmith.Agent is a modular autonomous agent framework that treats Minecraft (and other game worlds) as a world adapter while using the MemorySmith wiki as long-term memory. The design follows Domain-Driven, deep-module principles — each bounded context exposes a small stable interface and hides its complexity.

Agents build, explore, and survive in Minecraft using an HTN/GOAP hybrid planner backed by a searchable knowledge base of facts, blueprints, and observations.

Quick Start

git clone https://github.com/TheMasonX/MemorySmith.Agent
cd MemorySmith.Agent
dotnet restore MemorySmith.Agent.slnx
dotnet run --project WebUI.Blazor

Then visit /api/agent/status or send a goal:

curl -X POST http://localhost:5000/api/agent/plan \
  -H "Content-Type: application/json" \
  -d '{"goalName":"GatherWood","parameters":{"count":10}}'
